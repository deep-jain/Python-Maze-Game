# Python-Maze-Game
A simple maze game to demonstrate the turtle and tkinter modules. 

* Dependencies:  
    * Turtle
    * tkinter
* Testing and Run Enviroment:
    * OS: Ubuntu 20.10 x86_64
    * Command Lines: 
        * python3 maze.py
